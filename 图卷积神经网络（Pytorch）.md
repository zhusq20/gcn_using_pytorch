# README
图卷积神经网络（Pytorch）

## 配置

- PyTorch==1.8.1
- Python==3.8.8
- dgl==0.5.2
- scipy==1.5.2
- numpy==1.19.1
- networkx==2.5

## 使用方法

```python
python train.py --dataset cora
或
python train.py --dataset citeseer
```
